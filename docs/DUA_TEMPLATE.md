## Purpose

The recipient may use the controlled GBA-GCs components only for the approved non-commercial research purpose described in the access request.

## Restrictions

The recipient must not:

- redistribute raw data
- publish names, addresses, provider IDs, coordinates, AOI polygons, or raw image chips
- attempt resident-level or household-level identification
- use data for surveillance, targeting, policing, commercial real-estate decision-making, advertising, or geofencing
- combine controlled data with resident-level, household-level, phone, transaction, or mobility records
- transfer access to unauthorised users

## Security

The recipient must store files on access-controlled systems, restrict access to named project personnel, keep an access log, and delete or return the data at the end of the approved period if requested.

## Publication

Publications may report aggregate statistics and privacy-preserving visualizations. Per-compound sensitive attributes, precise geometries, and raw provider records must not be published.

## Breach And Misuse

The recipient must notify the data provider promptly of any suspected breach, accidental disclosure, or misuse.
